项目名称
Reverse TCP Client-Server

项目简介
这是一个简单的客户端-服务器程序，用于将客户端的文件内容发送到服务器，服务器将内容反转后返回给客户端，并将反转后的内容保存到新文件中。

运行环境
操作系统：Windows、Linux 或 macOS
Python 版本：Python 3.6 或更高版本

安装依赖
本项目无需额外安装第三方库，只需确保 Python 环境已安装即可。

配置选项
#客户端配置
serverName：服务器 IP 地址，需根据实际服务器地址进行配置。
serverPort：服务器端口号，默认为 8977，可根据需要修改。
Lmin 和 Lmax：报文块的最小长度和最大长度，根据实际需求设置合适的范围。
#服务器配置
serverPort：服务器监听端口号，默认为 8977，需与客户端配置一致。

运行步骤
#启动服务器
打开终端或命令提示符。
进入项目目录。
运行以下命令启动服务器：
python reversetcpserver.py
服务器将开始监听指定端口，等待客户端连接。
#启动客户端
1. 确保 `source.txt` 文件存在且非空，该文件将被发送到服务器进行处理。
2. 打开另一个终端或命令提示符窗口。
3. 进入项目目录。
4. 运行以下命令启动客户端：
python reversetcpclient.py
按提示输入服务器 IP 地址和端口号。
输入报文块的最小长度和最大长度。
客户端将读取文件内容，按指定长度分块发送到服务器，接收服务器返回的反转内容，并将反转后的内容保存到 reversedFile.txt 文件中。

注意事项
确保服务器和客户端的端口号一致。
客户端的 source.txt 文件需存在且非空，否则程序将退出。
根据网络环境和实际需求合理设置报文块的长度范围，避免过大或过小影响性能。

项目结构
reversetcpclient.py：客户端程序，负责读取文件内容、分块发送到服务器、接收反转内容并保存。
reversetcpserver.py：服务器程序，负责接收客户端连接、处理文件内容反转并返回结果。
